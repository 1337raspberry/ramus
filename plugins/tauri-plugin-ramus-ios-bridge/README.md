# Tauri Plugin ramus-ios-bridge

Internal Tauri plugin used by ramus to bridge into the iOS Swift layer (MPVKit playback, `MPRemoteCommandCenter`, native search bar) and the Android Kotlin layer (Media3 / ExoPlayer, `MediaSession`). Not intended for standalone use.

See the main project [README](../../README.md) and [CLAUDE.md](../../CLAUDE.md) for architecture notes.
